# 🎮 KBC Game (Kaun Banega Crorepati)

A Python-based quiz game inspired by the popular Indian TV show *Kaun Banega Crorepati (KBC)*.  
This project simulates a multiple-choice quiz game with increasing prize levels, lifelines, and a timer.

---

## 🚀 Features

- 🧠 Multiple-choice quiz questions
- 💰 Prize money system (like real KBC)
- ⏱️ Timer for answering questions
- 🎯 Lifelines (50-50, Audience Poll, etc.)
- 🔄 Randomized questions
- 💻 Console-based interactive gameplay

---

## 🛠️ Tech Stack

- Python
- Built-in libraries:
  - `random`
  - `time`
  - `threading`
  - `sys`
  - `os`
